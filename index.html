iRecommend

Type: MERN Stack App | Duration: 1 week | Project: Individual

I sometimes use tripadvisor when on holiday, although I never know which review to trust. They are always so different and when there is just one negative, you question yourself whether to go to that place at all. Therefore I decided to make an app where you are able to follow your friends/people that you trust and see their recommendations for each city - iRecommend. I used Google Autocomplete and Google Places API to find the place a user wants to recommend. I also used Rest Countries API to get the language and currency codes for each country where there were recommendations and then used Yandex, a translating API and Oanda, an exchange rate API to provide more information for users when they click on a certain city.

Evening Angel

Type: MEAN Stack App | Duration: 1 week | Project: Group


This was a group project creating a full MEAN stack app using our own API as well as others. For this app we wanted to make it easy for a user to bundle their evening together by giving them a selection of bars and restaurants near their chosen event; acting as a tool to organise their evening conveniently for them - Evening Angel. Firstly, we used Skiddle's API to return events so the user can find and select the event they are going to. We then used Google Places API to find nearby bars and restaurants in relation to the chosen event's location, within a radius chosen by the user. Once a 'bundle' is created, we provided a travel time calculation from a given starting location using Google Autocomplete and Citymapper's API.
